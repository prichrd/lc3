# LC-3
A [Little Computer 3](https://en.wikipedia.org/wiki/Little_Computer_3) VM written in go. 
Written to learn about CPUs and compiled programming languages.

## Installation
```bash
$ go get -u github.com/prichrd/lc3/cmd/lc3/...
```

## Usage
```
$ lc3 <image-path>
```
